# Editing & enriching

Everything the app generates is a **proposal, not a verdict**.

- **Edit** — click any box to change it. Your edits are kept and marked with a ✎, and survive
  re-reviews.
- **Enrich** — on several layers, Enrich proposes additions: either from common industry patterns
  (grounded) or from live **web research** (with source links). You keep only what fits — each suggestion
  is individually accepted or dropped, and **⚡ Auto** accepts them all.

Remove what doesn't belong, add what's missing — you're always in charge. See
[Protecting your fixes](../reviewing/protecting-your-fixes) for how these authored changes relate to
regeneration.
